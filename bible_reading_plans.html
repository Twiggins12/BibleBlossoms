Chronological Reading Plan

For a 2-yr. plan, start Day One with the first entry in Genesis. On Day Two, read the first entry from column two, Luke 1:1-25.

Each day you will alternate between the Old and New Testament.

For a 1-yr. plan, read one entry from each column daily. Days/dates are not included so that if you get behind, or want to read ahead, it won't matter. Simply add dates if you wish.
